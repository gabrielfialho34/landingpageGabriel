# Publicação da Landing

## Opção recomendada: GitHub Pages
1. Crie um repositório público no GitHub.
2. Envie todos os arquivos desta pasta para esse repositório.
3. Faça o push para a branch `main` ou `master`.
4. No GitHub, abra `Settings > Pages` e confirme `GitHub Actions` como fonte de publicação.
5. Aguarde o workflow `Deploy portfolio to GitHub Pages` concluir.
6. A URL pública ficará disponível no próprio GitHub Pages.

## Para aparecer no Google
1. Publique a página em uma URL pública estável.
2. Abra o Google Search Console.
3. Solicite a indexação da URL publicada.
4. Se usar domínio próprio, vale adicionar depois um `sitemap.xml` com a URL final.

## Observação importante
- O site já está pronto para deploy estático.
- Para SEO completo com `canonical` e `sitemap.xml`, falta apenas a URL final do site publicado.
